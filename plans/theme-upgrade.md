# Queen's Salon & Studio — Professional Theme Upgrade Plan

## Current State Assessment

The website has a solid foundation with:
- Clean layout structure across 7 pages
- Cormorant Garamond + Manrope font pairing
- Ivory/gold/blush color palette
- Masonry gallery with lightbox
- Scroll-reveal animations
- Responsive mobile menu

### Areas for Improvement
1. **Color palette feels flat** — the ivory/beige tones blend together without enough contrast
2. **Header lacks visual identity** — no logo icon, just text brand
3. **Service cards are plain** — no icons, minimal visual differentiation
4. **Gallery hover effects are basic** — just a scale transform
5. **Testimonials lack visual richness** — no star ratings, no avatar placeholders
6. **WhatsApp button shows "W" text** — needs proper icon
7. **Footer is minimal** — no social links, basic layout
8. **No decorative elements** — professional salon sites use gold dividers, ornamental accents
9. **Animations are basic** — only fade-up reveals, no shimmer or parallax

---

## Planned Changes

### 1. Color Palette Upgrade
```
--gold:        #c6a769  →  #b8860b (deeper, richer gold)
--gold-light:  NEW #d4af37 (bright gold for accents)
--rose-gold:   NEW #b76e79 (rose-gold accent)
--cream:       NEW #faf7f2 (warmer cream)
--deep:        NEW #0d0b09 (near-black for contrast)
--blush:       #e8cfc5  →  #e8d5c4 (warmer blush)
```

### 2. Header Enhancement
- Add a crown/tiara SVG icon next to brand name
- Improve glassmorphism with stronger blur and border
- Add gold underline animation on nav hover
- Slightly larger brand text with better weight contrast

### 3. Hero Section Redesign
- Add decorative gold corner ornaments
- Improve gradient overlay for better text readability
- Add subtle shimmer animation on the eyebrow text
- Better button styling with gold gradient and hover glow

### 4. Typography Improvements
- Keep Cormorant Garamond but add weight 400 for body quotes
- Increase heading sizes slightly for more drama
- Add gold color to eyebrow labels instead of grey
- Better letter-spacing on section headings

### 5. Decorative Elements
- Gold horizontal dividers between sections
- Ornamental SVG flourishes near section headings

- Gold border-left accents on blockquotes

### 6. Service Cards Enhancement
- Add beauty-related emoji/icons to each service card header
- Gold gradient top border on cards

- Better card spacing and padding

### 7. Gallery Improvements
- Gold border on hover for masonry items

### 8. Testimonial Section

### 9. CTA Section
- Add subtle pattern overlay on dark background

### 10. Footer Enhancement

### 11. WhatsApp Button

### 12. Animation Upgrades
- Gold shimmer keyframe animation for accents

### 13. Mobile Improvements
